Willkommen beim SymptomTracker!

Führen Sie ein Symptom-Tagebuch!

Der Symptom-Tracker ermöglicht Ihnen - als bereits positiv getesteter COVID-19 Patient oder Verdachtsfall - das Tracking Ihrer Covid-19-Symptome als Tagebuch. So erhalten Sie den Überblick über Ihren Krankheitsverlauf sowie Ihren aktuellen Gesundheitsstatus.

Sie haben einen Code vom Gesundheitsamt erhalten?

Wenn Sie nach COVID-19 Testung einen Code vom Gesundheitsamt erhalten haben, können Sie diesen hier eintragen und im Anschluss Ihr Symptomtagebuch führen.

Bald ist der SymptomTracker auch im Google Play Store und im App Store verfügbar!
